# Data

Локальные данные проекта. Содержимое этой папки игнорируется Git, кроме README и `.gitkeep`.

- `raw/` - исходные данные в максимально близком к выгрузке виде.
- `interim/` - промежуточные таблицы.
- `processed/` - подготовленные данные для моделей и пайплайнов.
- `samples/` - маленькие безопасные sample-файлы для тестов и демо.
