# Archive

Эта папка хранит старые EDA-материалы и код, который больше не является частью
production pipeline.

Код из `docs/archive/code/` нельзя импортировать из `src/`, `tests/` или
`scripts/` и нельзя использовать как runtime-зависимость.

Причины архивации:

- сохранить исторический контекст исследований;
- иметь reference для старых расчётов и обсуждений;
- не смешивать legacy EDA-код с production-слоями.
