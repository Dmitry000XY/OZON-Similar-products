# Legacy item popularity test example

Этот пример сохранён только как историческая заметка и не является
исполняемым тестом production-кода.

## Что здесь было

Раньше проверялся подход weighted item popularity для EDA-экспериментов, где
подсчёт шёл по прямым действиям с товаром и с event weights.

## Почему это теперь markdown, а не `.py`

- В production-пакете больше нет `features.eda_item_popularity`.
- Архивный пример не должен выглядеть как рабочий pytest-модуль.
- Для повторного использования логики её нужно переносить в `src/` отдельным PR
  с актуальными импортами и тестами.
